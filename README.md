# Hi there, I'm Alvin 👋

Fourth-year Informatics and Computer Science student at Strathmore University, Nairobi. I build across the stack, from mobile apps in Swift and Kotlin to backend systems in Python and Django, and I'm drawn to problems that sit at the edge of software and real-world infrastructure. Currently finishing my undergraduate thesis.

- 🔭 **Current Focus:** Huduma Ramani, a map-enabled community service directory for Nairobi's informal settlements, built with React Native, Next.js, Django REST Framework, and Firebase, with a hybrid ML recommendation engine.
- 🌱 **Learning & Exploration:** Embedded systems and IoT (ESP32, MicroPython, MQTT), distributed systems, and applied machine learning.
- ⚡ **Core Interests:** Mobile development, IoT and embedded systems, backend engineering, and community-centred technology.
